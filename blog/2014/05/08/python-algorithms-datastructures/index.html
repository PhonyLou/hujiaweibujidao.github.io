测试pop操作：从结果可以看出，pop最后一个元素的效率远远高于pop第一个元素

(2)Dictionary

Dictionary的各个操作的性能

Dictionary和List的性能比较：list基本上随着其元素的数目呈线性增长，而dictionary一直维持在很短很短的时间内(我的机子测试的结果都是0.001ms)。
